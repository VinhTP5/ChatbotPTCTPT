PHẦN A — TỔNG QUAN DỰ ÁN

Mục tiêu
--------
Chatbot RAG (Retrieval-Augmented Generation) hỏi-đáp về Chương trình Giáo dục
Phổ thông (PTCT PT), dựa trên:
  · Văn bản nội bộ FPT/FSC (Quyết định, Kế hoạch, Hướng dẫn) — thư mục data/QD/
  · Văn bản Nhà nước (Thông tư 32/2018, CT GDPT 2018, 27 môn học) — thư mục data/TT32_2018/

Stack chính
-----------
  · Frontend  : Streamlit (app.py)
  · RAG engine: LangChain + ChromaDB (src/)
  · Embedding : BGE-M3 (BAAI/bge-m3) — mặc định; hỗ trợ MiniLM, MPNet, E5
  · LLM       : đa provider — Groq, OpenAI, Anthropic, Google Gemini, DeepSeek
  · Chunking  : Late Chunking / Long Late Chunking / Standard
  · Reranker  : BGE-Reranker-v2-m3 (opt-in)
  · Deploy    : Streamlit Cloud (chroma_db được split thành .part.NNN + md5)
